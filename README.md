# Conversión mayúsculas.

Programa realizado en C++ el cual cambia el nombre de los diferentes directorios que se encuentren en la misma carpeta que el programa en cuestión al cambiar sus nombres de minúsculas a mayúsculas, esto con la particularidad de no hacer uso del paradigma de programación orientada a objetos.
